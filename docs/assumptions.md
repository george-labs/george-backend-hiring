# Assumptions per iteration

## Iteration 1

Words are separated by space, dot, comma, explanation mark, question mark, semicolon.
```shell
$ wordcount
Enter text: ""
Number of words: 0
$
```

```shell
$ wordcount
Enter text: Mary had a little lamb.
Number of words: 5
$
```

```shell
$ wordcount
Enter text: 1258
Number of words: 0
$
```

```shell
$ wordcount
Enter text: Mary had a li25le 1258 @! lamb. 
Number of words: 4
$
```
